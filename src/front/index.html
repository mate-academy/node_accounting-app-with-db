REST API with 2 endpoints

(using Node.js, Express, PostgreSQL)

All the requests can be test at BASE_URL:

https://accounting-app-with-db-production.up.railway.app/

Available resources:

/users /expenses

Method URL Description Request body Response data
GET /users Load all users - Collection
GET /users/2 Load a user by id - Single
POST /users Create a user { "name": "Your_name" } Single
PATCH /users/2 Update a user { "name": "New_name" } Single
DELETE /users/2 Delete a user - StatusCode: 204
