users/expenses API home page:

You can simply use this RESTFUL API by
sending a right http requests like (get, post, push and delete).
Here is some basic examples:

Users:

  • /users GET
  • /users/2 GET
  • /users POST
  • /users/2 DELETE
  • /users/2 PATCH

Expenses:

  • /expenses GET
  • /expenses/2 GET
  • /expenses POST
  • /expenses/2 DELETE
  • /expenses/2 PATCH

CRUD:

on Create you will get a new category,
on Read you will get all the categories,
on Update you will update the name BUT write it in form oldName-=-newName
on Delete you will delete the category
